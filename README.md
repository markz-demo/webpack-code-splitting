## 命令

首先安装最新稳定版本`nodejs` https://nodejs.org/en/download/

第一次运行`npm`装包指令：

`$ npm install`

开发本地服务模式：http://localhost:3000/

`$ npm run dev`

开发build：

`$ npm run build`

生产build：

`$ npm run prod`

webpack分析：

`$ npm run analy`
